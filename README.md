# Portfolio Web + Admin CMS

SPEC Reference: `SPEC.md`
Current Version: v1.2.0

## Setup

1. `npm install`
2. `cp .env.example .env.local` (Fill in values)
3. `npm run dev`

## Management

Admin Panel: `/admin`
(Login required, restricted to ADMIN_EMAIL)

## Tech Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Auth.js
- Supabase
